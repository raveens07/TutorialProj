Fundamentals

Hello, World!

A java program is made of classes, in other words a class is a fundamental unit of java program.Name of the java program should always be name of the class with ".java" extension. In the below code example name of the class is HelloWorld so java program should be named as HelloWorld.java. 

         
        
         	public class HelloWorld {
    			public static void main(String[] args) {
        			System.out.println("Hello, World!");
    			}
		}

Pointers from above code

  • main() method should always be public static void and must take single String array as argument
  • When a java program is executed JVM looks for main() method with signature "public static void main(String[] args)", it's like an entry point for the JVM
  • "System.out.println()" is used to display an output to the console

Basic Java glossary

Class
A class is blue print of real world entity from which objects are created. It defines what data an object can hold and what operations an object can do with the data.

Object
Object is an instance of class, it's a bundle of sate and behavior. Sate translates to data defined in the class and behaviors to the operations defined in the class.

Properties
These are the variables where class stores it's data, they define the sate of a class.

Methods
A Method has set of instructions to perform an operation, they define the behavior of class.

Package
A package is like folder where you place all your classes. It is useful for logical organization of all the classes defined in your project.
